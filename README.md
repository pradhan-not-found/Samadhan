## 🚀 What is Samadhan?

**Samadhan** (meaning *Resolution*) is a hyperlocal civic tech platform built to bring citizens and municipal authorities closer together. Fragmented reporting and untracked issues lead to public distrust. By combining gamification, predictive insights, and AI categorization, Samadhan transforms complaints into accountable community action.

Whether it's a broken streetlight, a dangerous pothole, or an overflowing garbage dump, Samadhan uses advanced AI to automatically tag, categorize, and route incoming reports to the correct government department instantly.

## ✨ Core Features

* 🧠 **AI Issue Categorization:** Automatically analyzes user reports (text and images), extracts keywords, determines severity, and assigns the correct primary category.
* 🔀 **Auto-Routing Engine:** Skips the bureaucratic red tape by routing verified issues directly to the relevant departments (e.g., PWD, Sanitation, Water Board).
* 🏆 **Community Gamification & Leaderboard:** Earn "Impact Points" for submitting valid reports and verifying community issues. Climb the leaderboard and become a Civic Hero.
* 📊 **Predictive Analytics Dashboard:** Deep data insights and resolution trend analysis to help authorities pre-emptively manage city infrastructure.

## 💻 Tech Stack

* **Frontend Framework:** React (Vite)
* **Styling:** Custom Vanilla CSS & CSS Variables (Dark/Light Mode Theme Support)
* **Animations:** Framer Motion
* **Typography:** Inter & Outfit (Google Fonts)

## 🛠️ Getting Started

To get a local copy up and running, follow these simple steps:

### Prerequisites
Make sure you have Node.js installed.

### Installation

1. Clone the repository
   ```sh
   git clone https://github.com/pradhan-not-found/Samadhan.git
   ```
2. Navigate into the project directory
   ```sh
   cd Samadhan
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Start the development server
   ```sh
   npm run dev
   ```
